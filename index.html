셀프 체크인/아웃 안내

• 오후 11시 이후 직원 부재

• 직원 지원 가능 시간: 오전 9시 ~ 오후 11시

• 입구 키오스크에서 셀프 체크인/아웃

체크인 방법

  1. 언어 선택
  2. "체크인" 선택
  3. 예약 검색:

    옵션 1: 이름으로 검색

    - 이름 또는 성 (4글자 이상)

    예시: 이름 - JOHN, 성 - SMITH

    옵션 2: 예약번호로 검색

    예시: Agoda - 1432779468

    Booking.com - 4617852306

  4. 확인 및 카드키 수령:

    1. 예약자명 확인

    2. "Next" 버튼 누르기

    3. 화면과 영수증의 객실 번호 확인

주의사항

• 카드키 분실 시 벌금: 10,000원

• 오후 11시 이후 분실 시 다음날 아침까지 체크인 불가

카드키 사용

• 입구 유리문 개폐용

• 객실 도어락 개폐용

    オプション1: 名前で検索

    例: 名 - JOHN, 姓 - SMITH

    オプション2: 予約番号で検索

  4. 確認とカードキー受け取り:

    2. "Next"ボタンを押す

    3. 画面とレシートの部屋番号確認
